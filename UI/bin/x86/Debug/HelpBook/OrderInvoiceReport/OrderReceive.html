Order Stock Report Setup


  • Reports အောက်ရှိ Order Stock Form ကိုဖွင့်ပါ။
  • Order Stock Report Form သည် အော်ဒါအပ် ပစ္စည်းစာရင်းများ နှင့် အော်ဒါပြန်ရွေး ပစ္စည်းစာရင်းများကို ပြန်လည်ကြည့်နိုင်သော Form ဖြစ်သည်။ အော်ဒါလက်ခံခြင်းအတွက်သာ ကြည့်လိုပါက အော်ဒါ Button အားရွေးချယ်ကြည့်ရှုနိုင်သည်။
  • အော်ဒါပစ္စည်းစာရင်းချုပ် ကို ကြည့်လိုပါက စာရင်းချုပ် ကို on ထားပါ။
  • အော်ဒါလက်ခံခြင်းအား အသေးစိတ် ကြည့်လိုပါက အသေးစိတ် ကို on ထားပါ။ အသေးစိတ် ကြည့်ရှုရာတွင် အော်ဒါလက်ခံရာ၌ ဖြည့်စွက်ထားသော Data များကို ကြည့်ရှုနိုင်မည်ဖြစ်သည်။
  • မိမိကြည့်လိုသော နေ့စွဲ ကိုရွေးချယ်ပေးရပါမည်။ ရွေးချယ်ထားသော နေ့စွဲ အတွင်းရှိသော Data များကို တွေ့ရမည် ဖြစ်သည်။
  • ဝယ်သူ အလိုက်ကြည့်လိုပါက ဝယ်သူ၏ check box ကို on ထားပြီး ဝယ်သူ၏ ဘေးတွင်ရှိသော မှန်ဘီလူး button ကိုနှိပ်ပြီး ကြည့်လိုသော ဝယ်သူကို ရွေးချယ် ပြီး ၎င်းနှင့်ပတ်သက်သော Data များကိုကြည့်နိုင်ပါသည်။ (သို့) ရွှေရည် အလိုက်ကြည့်လိုပါက ရွှေရည် ၏ check box ကို on ထားပေးပါ။ (သို့) ပစ္စည်းအမျိုးအစား အလိုက်ကြည့်လိုလျှင် ပစ္စည်းအမျိုးအစား၏ check box ကို on ထားပေးပါ။ (သို့) သုံးမျိုးလုံး ကြည့်လိုပါက ဝယ်သူ၊ ရွှေရည် နှင့် ပစ္စည်းအမျိုးအစား သုံးမျိုးလုံးကို on ထားပြီး Data များကိုပြန်လည်ကြည့်နိုင် ပါသည်။
  • Cancel checkbox အား on ပြီး View Button နှိပ်ပါက Cancel လုပ်ထားသော Data များသာ တွေ့ရမည်ဖြစ်သည်။ ၎င်း Check Box သည် Return အားရွေးချယ်ထားပါက မှိန်နေမည်ဖြစ်သည်။
  • Order By Due Date check box ကို on ထားပါက အော်ဒါရွေးရန်ကျန်နေသော ပစ္စည်းများကို တွေ့ရမည်ဖြစ်သည်။
  • Order Stock Form အား အသုံးပြုပြီးပါက Close Button ကိုနှိပ်၍ ပိတ်နိုင်ပါသည်။
