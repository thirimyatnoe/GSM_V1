Consignment Sale


  • Transaction အောက်ရှိ Consignment Sale ကိုဖွင့်ပါ။
  • Consignment ID တွင် Series အလိုက် Software မှ Auto တွက်ပေးပါမည်။
  • Date တွင် ယခု ရောက်ရှိနေသော အချိန်ကို ဖော်ပြပေးပါမည်။
  • WholeSales ID ဘေးရှိ မှန်ဘီလူးကိုနှိပ်၍ Consignment Sale ရောင်းရန် WholeSales တွင် Consignment ဖြင့် ဖွင့်ခဲ့သော WholeSales Voucher များနှင့် အတူ wholeSale Return ပြန်ချပြီး ကျန်နေသော အထည်များကျလာမည်ဖြစ်သည်။
  • Consignment Sale ကို လဲခြင်းဖြင့်ချိတ်ဆက်ထားသော အ၀◌ယ်များနှင့်လည်း တွဲပြီးဘောင်ချာဖွင့်နိုင်ပါသည်။
  • Data များ ထည့်သွင်းပြီးပါက Save Button ကို နှိပ်၍ သိမ်းဆည်းနိုင်ပါသည်။
  • အသစ်ပြုလုပ်မည်ဆိုပါက New Button ကိုနှိပ်၍ အသစ်ပြုလုပ်နိုင်ပါသည်။
  • ပြန်လည်ကြည့်ရှုလိုပါက Consignment ID ဘေးရှိ မှန်ဘီလူးကိုနှိပ်၍ ကြည့်နိုင်ပါသည်။
  • Data များပြန်ဖျက်လိုပါက Consignment ID ဘေးရှိ မှန်ဘီလူးကိုနှိပ်၍ ဖျက်လိုသော Voucher ကို Select မှတ်ပြီး Delete Button ကို နှိပ်၍ ဖျက်နိုင်ပါသည်။
  • Consignment Form အသုံးပြုပြီးပါက Close Button ကို နှိပ်၍ ပိတ်နိုင်ပါသည်။
