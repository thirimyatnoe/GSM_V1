Location Setup


  • Master Setup အောက်မှ Location Form ကိုနှိပ်ပါ။
  • ဆိုင်အမည်၊ လိပ်စာ၊ ဖုန်းနံပါတ်၊ Remark 1, Remark 2 တို့ကိုဖြည့်စွက်ပါ။
  • Remark 1 နှင့် Remark 2သည် Voucher ထုတ်ရာတွင် Voucher ၏အောက်ခြေတွင်ပေါ်မည့် စာသားများ ဖြစ်သည်။
  • ဆိုင်ခွဲများရှိပါက သက်ဆိုင်ရာဆိုင်ခွဲများအလိုက်ရွေးပြီး Set This PC As Choose Current Location ကိုနှိပ်ပါ။
  • Data များဖြည့်စွက်ပြီးပါက Save button ကိုနှိပ်ပြီးသိမ်းဆည်းရပါမည်။
