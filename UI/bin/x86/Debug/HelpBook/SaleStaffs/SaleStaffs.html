Sale Staffs Setup


  • Master Setup အောက်ရှိ Staff Form ကိုနှိပ်ပါ။
    • ဝန်ထမ်းအမည်များအား English (သို့) မြန်မာ3 font နှင့် ရိုက်ထည့်နိုင်ပါသည်။
  • ဖြည့်စွက်ထားသော ဝန်ထမ်းအမည်အား ပြင်လိုပါက ပြင်လိုသော ဝန်ထမ်းအမည် နေရာတွင် click ထောက်ပါ။ အမည်အားလုံးကို select မှတ်ထားသည့်ပုံစံဖြစ်သွားလျှင် ပြန်လည်ပြင်ဆင်နိုင်ပါသည်။
  • ဝန်ထမ်းအမည်ကို ဖျက်လိုပါက ဖျက်လိုသော ဝန်ထမ်းအမည်၏ရှေ့ဆုံးအကွက်ကိုထောက်ပြီး Computer တွင်ပါရှိသော Delete Button ကိုနှိပ်ပြီး ဖျက်ပါ။
  • ပြီးလျှင် Save button ကိုနှိပ်ပြီး သိမ်းဆည်းရပါမည်။
  • အကယ်၍ မိမိဖျက်လိုက်သော ဝန်ထမ်းအမည်သည် Transaction အောက်ရှိ Form တစ်ခုခု တွင် အသုံးပြုပြီးသားအမည် ဖြစ်ပါက "You can not delete this Staff " ဟူသော Massage Box တစ်ခုကျလာပြီး ဖျက်ခွင့်ပေးမည် မဟုတ်ပါ။
  • Sale Staffs Form အား အသုံးပြုပြီးပါက Close Button ကိုနှိပ်၍ ပိတ်နိုင်ပါသည်။
