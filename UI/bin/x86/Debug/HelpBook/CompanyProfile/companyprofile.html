Company Profile Setup


  • System Utilities အောက်မှ Company Profile Form ကိုဖွင့်ပါ။
  • Company Name - အသုံးပြုသူ၏ Company Name အား Software မှ Auto ထည့်ပေးမည် ဖြစ်သည်။
  • Phone, Email, Website, Fax Number, Address တို့ကိုဖြည့်စွက်ပေးပြီး Company Logo တွင် Add Button ကိုနှိပ်၍ Company Logo ထည့်နိုင်ပါသည်။
  • Data များဖြည့်စွက်ပြီးပါက Save Button ကိုနှိပ်၍ သိမ်းဆည်းနိုင်ပါသည်။
  • Company Profile Form ကိုအသုံးပြုပြီးပါက Close Button ကိုနှိပ်၍ ပိတ်နိုင်ပါသည်။
