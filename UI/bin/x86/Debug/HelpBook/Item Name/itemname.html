Item Name Setup


  • Master Setup အောက်မှ Item Name Form ကိုဖွင့်ပါ။
  • ပစ္စည်း အမျိုးအစား တစ်ခုပေါ်မူတည်ပြီး အမျိုးအမည် သတ်မှတ်နိုင်သော Form ဖြစ်ပါသည်။
  • ပစ္စည်း အမျိုးအစား ကို ရွေးချယ်ပါ။ ထို့နောက် Item Name များကို English (သို့) မြန်မာ3 font ဖြင့်ဖြည့်စွက် နိုင်ပါသည်။
  • ဖြည့်စွက်ထားသော Item Name အား ပြင်လိုပါက ပြင်လိုသော Item Name နေရာတွင် click ထောက်ပါ။ Item Name အားလုံးကို select မှတ်ထားသည့်ပုံစံဖြစ်သွားလျှင် ပြန်လည်ပြင်ဆင်နိုင်ပါသည်။
  • Item Name ကို ဖျက်လိုပါက ဖျက်လိုသော Item Name ၏ရှေ့ဆုံးအကွက်ကိုထောက်ပြီး Computer တွင်ပါရှိသော Delete Button ကိုနှိပ်ပြီး ဖျက်ပါ။
  • Data များ ဖြည့်သွင်းပြီးပါက Save button ကိုနှိပ်ပြီးသိမ်းဆည်းရပါမည်။
  • အကယ်၍ မိမိဖျက်လိုက်သော Item Name သည် Transaction အောက်ရှိ Form တစ်ခုခု တွင် အသုံးပြုပြီးသား ဖြစ်ပါက" You can not delete at row 1. This Item Name which has Transaction Recores!" ဟူသော Massage Box တစ်ခုကျလာပြီး ဖျက်ခွင့်ပေးမည် မဟုတ်ပါ။
  • Item Name Form အား အသုံးပြုပြီးပါက Close Button ကိုနှိပ်၍ ပိတ်နိုင်ပါသည်။
