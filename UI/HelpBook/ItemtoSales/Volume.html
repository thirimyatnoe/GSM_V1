ItemToSale Setup(Order)

  • Item to Sale တွင် volume အတွက် Barcode သတ်မှတ်လိုလျှင် Item to Sale Form ရှိ Volume check box တွင် အမှန်ခြစ်ခြစ်ပေးပါက Volume Stock Form ပေါ်လာမည်။
  • Volume Stock ဆိုသည်မှာ အသေးစားအထည်များကိုအထုပ်တစ်ထုပ်ထဲတွင်စုထည့်ပြီး ဘားကုတ်တစ်ခုတည်း သတ်မှတ်ထားခြင်းဖြစ်ပါသည်။ဥပမာ... တစ်ဖက်နားကပ်၊ကလေးလက်စွပ်၊ကလေးလက်ကောက်... စသည်ဖြင့်
  • Stock # တွင် Stock ID ကို Serial အလိုက် Auto တပ်ပေးပါမည်။
  • နေ့စွဲတွင် လက်ရှိရောက်နေသော Date ကိုပြပေးနေမည်ဖြစ်သည်။
  • တာဝန်ခံဝန်ထမ်းကိုရွေးပေးရပါမည်။
  • အမျိုးအစားကို ရွေးချယ်ပါ။ ထိုသို့ရွေးချယ်လိုက်သည်နှင့် Barcode No တွင် Barcode ကို Auto တပ်ပေးပါမည်။ အမျိုးအမည်၊ အရေအတွက်၊ ရွှေရည်၊ ကြိုးရှည်ကွင်းတိုင်း စသည်တို့ကိုဖြည့်စွက်ပါ။ Fix Price နှင့် ရောင်းချမည်ဆိုပါက Fix Selling Price(For One Item) check box ကို အမှန်ခြစ် ခြစ်ထားပေးပြီး သတ်မှတ်ဈေးနှုန်းကို ထည့်ပေးရပါမည်။ ထိုဈေးနှုန်းသည် အထုပ်ထဲမှ တစ်ခုချင်းစီကိုရောင်းမည့်ဈေးဖြစ် သောကြောင့် ဈေးတူသော အထည်များကိုသာထည့်ထားသင့်ပါသည်။ ရွှေရည်ပေါ်မူတည်ပြီး ပေါက်ဈေးဖြင့်ရောင်းမည်ဆိုပါက အမှန်ခြစ် မခြစ်ရပါ။
  • စုစုပေါင်းချိန်ကို ဖြည့်စွက်ပါက ရွှေရည်တွင် ကျပ်သားချိန်ကို ရွေးထားပါက အလေးချိန်တွင် သတ်မှတ်ရာတွင် K/P/Y ကိုသာရိုက်ထည့်နိုင်မည်ဖြစ်ပြီး Gram ချိန်ရွေးထားပါက gram အလေးချိန် ကိုသာ ရိုက်ထည့် နိုင်မည် ဖြစ်သည်။ ဝယ်ရင်းဈေးတန်ဖိုးကိုသတ်မှတ်လိုလျှင် ဝယ်ရင်းဈေးcheck box တွင် အမှန်ခြစ်ခြစ်ပေးပြီး ဈေးနှုန်း ကိုဖြည့်စွက်ပေးရပါမည်။ ထိုဝယ်ရင်းဈေးနှုန်းကိုဖြည့်စွက်ရာ၌ အထုပ်တစ်ခုလုံး၏ ဝယ်ရင်းတန်ဖိုးကို တွက်ချက်ပြီး ထည့်‌ပေးထားရပါမည်။
  • Solid Volume ကို အမှန်ခြစ်ထားပါက အထည်များအားအလေးချိန်အလိုက်ခွဲရောင်းနိုင်မည်ဖြစ်သည်။( Sale By Weight)
  • Solid Volume ကို Checkbox off ထားပါက အထည်များအား တစ်ခုချင်းစီ အရေအတွက်လိုက်ရောင်းနိုင်မည်ဖြစ်သည်။ (Sale by Qty and Weight)
  • Volume Stock ကို အရှုံး၊အမြတ်တွက်ရာတွင် ထိုအထုပ်ထဲမှအထည်များအားလုံးရောင်းပြီးမှ အရှုံး၊အမြတ် စာရင်း တွက်ပေးပါမည်။ (Reports -> Profit for Volume Items)

  • Photo ကိုထည့်သွင်းလိုပါက Add Button ကိုနှိပ်၍ Photo ထည့်သွင်းနိုင်သည်။
  • Data များ ဖြည့်စွက်ပြီးပါက Save Button နှိပ်၍ သိမ်းဆည်းနိုင်ပါသည်။
  • သိမ်းဆည်းပြီးသား Data များအား ပြန်လည်ကြည့်ခြင်း၊ ပြင်ဆင်ခြင်းများပြုလုပ်ချင်ပါက Stock # ၏ ဘေးတွင် ရှိသော မှန်ဘီလူး button ကိုနှိပ်ပြီးပြန်လည်ကြည့်ရှုပြင်ဆင်ချင်သော Sale Item အား ရွေးချယ် ပြီးပြင်နိုင် ပါသည်။
  • Barcode ထုတ်ချင်လျှင် Print Button ကိုနှိပ်၍ ထုတ်နိုင်ပါသည်။
  • သိမ်းဆည်းပြီးသား ထည့်ပြီးသား Data များကို ဖျက်လိုလျှင် မှန်ဘီလူး button အားနှိပ်ပြီးရှာပါ။ ထို့နောက် မိမိဖျက်လိုသော Data ကို Delete Button ကိုနှိပ်၍ ဖျက်နိုင်ပါသည်။
  • Sale Item အသစ်ထည့်လိုပါက New Button ကိုနှိပ်ပြီး အသစ်ထည့်နိုင်ပါသည်။
  • Item to Sales Form အား အသုံးပြုပြီးပါက Close Button ကိုနှိပ်၍ ပိတ်နိုင်ပါသည်။
