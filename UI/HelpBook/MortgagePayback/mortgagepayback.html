Mortgage Payback


  • အရင်းငွေ ဆပ်ရန်အတွက် Transaction အောက်ရှိ Mortgage Loan Payback Form ကိုဖွင့်ပါ။
  • Payback ID တွင် Serial အလိုက် Software မှ Auto ကျပေးပါသည်။
  • အပေါင်နံပတ်ကိုရွေးချယ်လိုက်သည်နှင့် Mortgage Invoice ID၊ နေ့စွဲ၊ အပေါင်တာ၀န်ခံ၊ အတိုးနှုန်း၊ စုစုပေါင်းပေးငွေ၊ ပေါင်သူအမည်၊ လိပ်စာ တို့သည် Mortgage Receive တွင်သတ်မှတ်ထားသည့်အတိုင်း Auto ကျလာမည်။
  • Grid ထဲရှိ Payback Amout တွင် Item တစ်ခုချင်းစီ၏ ဆပ်လိုသော အရင်းငွေပမာဏကိုဆပ်နိုင်ပါသည်။ Item တစ်ခုချင်းစီ၏ Payback Amount ကိုထည့်သွင်းလိုက်ပါကTotal Payback Amount တွင် စုစုပေါင်း Payback Amount ကိုပြပေးပါသည်။
  • ပေးသည့်နေ့စွဲတွင် လက်ရှိရောက်နေသော Current Date ကိုပြပေးပါသည်။
  • နေ့စွဲ (မှ) တွင် အပေါင်လက်ခံသည့် ရက်စွဲ (သို့မဟုတ်) အတိုးစတွက်သည့်ရက်စွဲ ကိုပြပေးပါသည်။နေ့စွဲ အထိ တွင် အတိုးတွက်မည့်နေ့စွဲ (သို့မဟုတ်)Current Date ကိုပြပေးပါသည်။
  • ပေးငွေ Discount၊ မှတ်ချက် တို့ကိုဖြည့်စွက်ပါ။
  • အချက်အလက်များအားလုံးဖြည့်စွက်ပြီးပါက Save Button ကိုနှိပ်၍ သိမ်းဆည်းထားနိုင်ပါသည်။
  • သိမ်းဆည်းပြီးသား Data များကိုဖျက်လိုလျှင် Payback ID ဘေးရှိ မှန်ဘီလူးကိုနှိပ်၍ ဖျက်လိုသော Vocher ကိုရွေးချယ်ပြီး Delete Button ကိုနှိပ်၍ ဖျက်နိုင်ပါသည်။
  • Mortgage Payback အသစ်သွင်းလိုပါက New Button ကိုနှိပ်၍ သွင်းနိုင်ပါသည်။
  • Mortgage Loan Payback Form ကိုအသုံးပြုပြီးပါက Close Buttonကိုနှိပ်၍ ပိတ်နိုင်ပါသည်။
