Sale Stock(Loose Diamond)


  • Sale Loose Diamond Invoice ဖွင့်ရန်အတွက်Transaction အောက်မှ Sale Stock(Loose Diamond) Form ကို ဖွင့်ပါ။
  • Sale Voucher No. ကို Software မှ Auto တပ်ပေးပါမည်။ နေ့ စွဲတွင် လက်ရှိရောက်နေသော Date ကိုပြပေးမည် ဖြစ်သည်။
  • တာဝန်ခံ ဝန်ထမ်းကို ရွေးချယ်ပါ။ ဝယ်သူ၏ code နံပါတ်ကိုအလွတ်သိပါကရိုက်ထည့်လျှင် ဝယ်သူအမည်နှင့် လိပ်စာကျလာပါမည်။နံပါတ်ကို အလွတ်မသိရှိပါက ဝယ်သူဘေးတွင်ရှိသောမှန်ဘီလူးကိုနှိပ်ပြီး ရွေးချယ်ရပါမည်။
  • ဘားကုတ်နံပါတ်ကို Barcode Scanner ဖြင့်လဲဖတ်နိုင်ပါသည်။ သို့မဟုတ် Barcode No ဘေးတွင် ရှိသော မှန်ဘီလူး button ကိုနှိပ်၍ ‌ရောင်းချမည့် Barcode ကိုရွေးချယ်နိုင်ပါသည်။
  • Barcode ကိုရွေးချယ်လိုက်သည်နှင့် တပြိုင်နက် Stock Setup Form (Loose Diamond)တွင် သတ်မှတ်ခဲ့သည့် အတိုင်း စိန်အမျိုးအစား၊ original code Color Clarity တို့ ပေါပေးမည် ဖြစ်ပြီး ပေါက်စေ◌ျးသည် Define Diamond Price ရှိ ကာရက်ပေါမူတည်ပြီး စေ◌ျးကျပေးမည် ဖြစ်သည်။
