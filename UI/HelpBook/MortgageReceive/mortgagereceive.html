Mortgage Receive


  • Transaction အောက်ရှိ Mortgage Receive Form သည်အပေါင်ပစ္စည်းများလက်ခံရန်အတွက် စာရင်းသွင်းနိုင်သော Form ဖြစ်ပါသည်။
  • Mortgage Receive ID သည် System Utilities Menu အောက်ရှိ Generate Format တွင် သတ်မှတ်ထားသော MortgageReceive Format အတိုင်းကျလာမည်ဖြစ်သည်။
  • အပေါင်နံပတ်သည် System Utilities Menu အောက်ရှိ Generate Format တွင် သတ်မှတ်ထားသော Mortgage Invoice Code Format အတိုင်းကျလာမည်ဖြစ်သည်။
  • နေ့စွဲတွင် လက်ရှိရောက်နေသော Current Date ကိုပြပေးပါသည်။
  • တာ၀န်ခံ၀န်ထမ်း၊ မှတ်ချက်၊ ၀ယ်သူတို့ကိုဖြည့်စွက်ပါ။
  • အတိုးနှုန်း နှင့် အပေါင်ဆုံးကာလတို့သည် Global Setting အောက်ရှိ Mortgage Rate နှင့် Mortgage Period တို့တွင်သတ်မှတ်ထားသည့်နှုန်းအတိုင်း ဖော်ပြပေးပါသည်။
  • ရွှေရည်၊ အမျိုးအစား၊ အမျိုးအမည်တို့ကိုဖြည့်စွက်ပါ။ ပေါက်စျေးသည် ရွှေရည်ကိုရွေးလိုက်သည်နှင့် Master Setup အောက်ရှိ Define Gold Price တွင်သတ်မှတ်ခဲ့သည့် ပေါက်စျေးအတိုင်း Auto ကျလာမည်။
  • ရွှေအလေးချိန်(စက်ချိန်)တွင်အထည်ချိန်ကိုထည့်လိုက်သည်နှင့်တစ်ပြိုင်နက်ခန့်မှန်း‌ငွေပမာဏ Autoကျလာ မည်။
  • ချေးငွေကျသင့်ငွေ(%)တွက် ရန်အတွက် Percent ကိုနှိပ်၍ ရာခိုင်နှုန်းအလိုက်သတ်မှတ်နိုင်ပါသည်။ ရာခိုင်နှုန်း သတ်မှတ်ရန်အတွက် Percent ကိုနှိပ်လိုက်လျှင် Keyword Form တစ်ခုကျလာမည်။ Keyword Name ဘေးရှိ ComboBox တွင် DonePercent ကိုရွေးပြီး ရာခိုင်နှုန်းသတ်မှတ်နိုင်ပါသည်။
  • ပေးငွေ ပမာဏကိုဖြည့်စွက်ပါ။ထို့‌နောက် Add Buttoon ကိုနှိပ်၍ စာရင်းသွင်း နိုင်ပါသည်။ (ဥပမာ - Vocher တစ်စောင်တွင် အပေါင်ပစ္စည်းသုံးမျိုး လက်ခံမည်ဆိုပါက သက်ဆိုင်ရာ အချက်အလက်များကို တစ်ခုချင်းစီဖြည့်သွင်းပြီး Add Button ကိုနှိပ်၍ စာရင်းသွင်း နိုင်ပါသည်။) Add လုပ်ပြီးသား ပစ္စည်းတစ်ခုကို ပြန်လည်ပြင်ဆင်လိုပါက Grid မှ မိမိပြင်ဆင်လိုသော Row ကိုနှိပ်လိုက်ပါက သက်ဆိုင်ရာ Row ၏ Data များကိုမူလစာရင်းသွင်းခဲ့သည့် နေရာတွင် ပြန်လည် ပြင်ဆင် နိုင်မည် ဖြစ်သည်။
  • စုစုပေါင်းအလေးချိန်တွင် Grid ထဲတွင် စာရင်းသွင်းခဲ့သော ပစ္စည်းအားလုံးပေါင်း၏ စုစုပေါင်း အလေးချိန်ကို ပြပေးပါသည်။ စုစုပေါင်းပေးငွေ တွင် Grid ထဲတွင် စာရင်းသွင်းခဲ့သော ပစ္စည်းအားလုံးပေါင်း၏ စုစုပေါင်း ပေးငွေကို ပြပေးပါသည်။
  • အချက်အလက်များအားလုံးဖြည့်စွက်ပြီးပါက Save Button ကိုနှိပ်၍သိမ်းထားနိုင်ပါသည်။ Save Button ကိုနှိပ်လိုက်သည်နှင့် “Do You Want To Print Mortgage Invoice” ဟူသော Message Box ကျလာမည်။ OK Button ကိုနှိပ်လိုက်ပါက Vocher တစ်ခါထဲထွက်လာမည်။ Cancel Button ကိုနှိပ်လိုက်ပါက အချက်လက်များကိုသိမ်းဆည်းပေးသွားမည်ဖြစ်သည်။
  • သိမ်းဆည်းပြီးသား Data များကိုဖျက်လိုလျှင် Mortgage Invoice ID ဘေးရှိ မှန်ဘီလူးကိုနှိပ်၍ ဖျက်လိုသော Vocher ကိုရွေးချယ်ပြီး Delete Buttoon ကိုနှိပ်၍ ဖျက်နိုင်ပါသည်။
  • Mortgage Receive Item အသစ်သွင်းလိုပါက New Button ကိုနှိပ်၍ စာရင်းသွင်းနိုင်ပါသည်။
  • Mortgage Receive Formအားအသုံးပြုပြီးပါက Close Button ကိုနှိပ်၍ ပိတ်နိုင်ပါသည်။
