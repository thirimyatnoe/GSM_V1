Mortgage Return


  • Transaction အောက်ရှိ Mortgage Returnသည် အပေါင်ပစ္စည်းရွေးသော Form ဖြစ်ပါသည်။
  • Return ID ကို Serial အလိုက် Software မှ Auto ကျပေးပါသည်။
  • အပေါင်နံပတ်ကိုရွေးချယ်လိုက်သည်နှင့် Mortgage Receive တွင်သတ်မှတ်ခဲ့သော အချက်အလက်များအားလုံး Auto ကျလာမည်။
  • Mortgage Item အောက်ရှိ Grid တွင် Return Item များကိုရွေးချယ်နိုင်ပါသည်။ စက်ချိန်တို့ကိုဖြည့်စွက်ပါ။
  • Interest And Payback History တွင် အတိုး‌‌ဆပ်ခဲ့သော ပမာဏနှင့် Payback History တို့ကိုပြန်လည်ကြည့်ရှု့နိုင်ပါသည်။
  • ပစ္စည်းရွေးသည့်နေ့စွဲတွင် လက်ရှိရောက်နေသော Current Date ကိုပြပေးပါသည်။
  • နေ့စွဲ (မှ) တွင် အပေါင်လက်ခံသည့် ရက်စွဲ (သို့မဟုတ်) အတိုးစတွက်သည့်ရက်စွဲ ကိုပြပေးပါသည်။နေ့စွဲ (အထိ) တွင် အတိုးတွက်မည့်နေ့စွဲ (သို့မဟုတ်)Current Date ကိုပြပေးပါသည်။
  • Mortgage Return Item ကိုရွေးချယ်လိုက်သည်နှင့် အတိုးတွက်သောကာလ၊ အတိုးကျသင့်‌ငွေ၊ အသာတင်ကျသင့်ငွေ Auto ကျလာမည်ဖြစ်သည်။ Mortgage Return Item သည် Calculate Mortgage Interest and Mortgage Payback တို့တွင် တစ်ခါထဲပေးချေပါက Mortgage Return တွင် အတိုးတွက်သောာကာလ၊ အတိုးကျသင့်ငွေ၊ အသာတင်ကျသင့်ငွေ ကျလာမည်မဟုတ်ပေ။
  • အချက်အလက်များအားလုံးဖြည့်စွက်ပြီးပါက Save Button ကိုနှိပ်၍ သိမ်းဆည်းထားနိုင်ပါသည်။ Save Button ကိုနှိပ်လိုက်သည်နှင့် “Do You Want To Print Mortgage Return Invoice” ဟူသော Message Box ကျလာမည်။ OK Button ကိုနှိပ်လိုက်ပါက Vocher တစ်ခါထဲထွက်လာမည်။ Cancel Button ကိုနှိပ်လိုက်ပါက အချက်လက်များကိုသိမ်းဆည်းပေးသွားမည်ဖြစ်သည်။
  • သိမ်းဆည်းပြီးသား Data များကိုဖျက်လိုလျှင် Return ID ဘေးရှိ မှန်ဘီလူးကိုနှိပ်၍ ဖျက်လိုသော Vocher ကိုရွေးချယ်ပြီး Delete Button ကိုနှိပ်၍ ဖျက်နိုင်ပါသည်။
  • Mortgage Return အသစ်သွင်းလိုပါက New Button ကိုနှိပ်၍ သွင်းနိုင်ပါသည်။
  • Mortgage Return Form ကိုအသုံးပြုပြီးပါက Close Buttonကိုနှိပ်၍ ပိတ်နိုင်ပါသည်။
