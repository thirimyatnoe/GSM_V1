Order Stock Report (Order Stock Return Report)


  • Reports အောက်ရှိ Order Stock Form ကိုဖွင့်ပါ။
  • Order Stock Report Form သည် အော်ဒါအပ် ပစ္စည်းစာရင်းများ နှင့် အော်ဒါပြန်ရွေး ပစ္စည်းစာရင်းများကို ပြန်လည်ကြည့်နိုင်သော Form ဖြစ်သည်။ အော်ဒါပြန်ရွေးခြင်းအတွက် ကြည့်လိုပါက အော်ဒါပြန်ရွေးButton အားရွေးချယ်ကြည့်ရှုနိုင်သည်။
  • အော်ဒါပြန်ရွေး ပစ္စည်းစာရင်းချုပ် ကို ကြည့်လိုပါက စာရင်းချုပ်ကို on ထားပါ။
  • အော်ဒါပြန်ရွေးပစ္စည်းစာရင်း ကို ကြည့်လိုပါက အသေးစိတ် ကို on ထားပါ။ အသေးစိတ် ကြည့်ရှုရာတွင် အော်ဒါရွေးရာ၌ ဖြည့်စွက်ထားသော Data အသေးစိတ်များကို ကြည့်ရှုနိုင်မည်ဖြစ်သည်။
  • မိမိကြည့်လိုသော နေ့စွဲ ကိုရွေးချယ်ပေးရပါမည်။ ရွေးချယ်ထားသော နေ့စွဲ အတွင်းရှိသော Data များကို တွေ့ရမည် ဖြစ်သည်။
  • ဝယ်သူ အလိုက်ကြည့်လိုပါက ဝယ်သူ၏ check box ကို on ထားပြီး ဝယ်သူ၏ ဘေးတွင်ရှိသော မှန်ဘီလူး button ကိုနှိပ်ပြီး ကြည့်လိုသော ဝယ်သူကို ရွေးချယ် ပြီး ၎င်းနှင့်ပတ်သက်သော Data များကိုကြည့်နိုင်ပါသည်။ (သို့) ရွှေရည် အလိုက်ကြည့်လိုပါက ရွှေရည် ၏ check box ကို on ထားပေးပါ။ (သို့) ပစ္စည်းအမျိုးအစား အလိုက်ကြည့်လိုလျှင် ပစ္စည်းအမျိုးအစား၏ check box ကို on ထားပေးပါ။ (သို့) သုံးမျိုးလုံး ကြည့်လိုပါက ဝယ်သူ၊ ရွှေရည် နှင့် ပစ္စည်းအမျိုးအစား သုံးမျိုးလုံးကို on ထားပြီး Data များကိုပြန်လည်ကြည့်နိုင် ပါသည်။
  • Order Stock Form အား အသုံးပြုပြီးပါက Close Button ကိုနှိပ်၍ ပိတ်နိုင်ပါသည်။
