Mortgage Disable


  • Transaction အောက်ရှိ Mortgage Disable သည် အပေါင်ဆုံးပစ္စည်းစာရင်းများကို Date အလိုက်ကြည့်ရှု့နိုင်သော Form ဖြစ်ပါသည်။
  • အပေါင်ဆုံးနေ့စွဲ၊ အပေါင်ဆုံးကာလ တို့ကို့ထည့်သွင်းပါ။ ထို့နောက် အပေါင်ဆုံးစာရင်း Button ကိုနှိပ်လိုက်သည်နှင့် Grid ထဲတွင် အပေါင်ဆုံးစာရင်းများကျလာမည်။
  • အပေါင်ဆုံးသတ်မှတ်လိုသော Item ကိုရွေးချယ်ပြီး Grid ထဲရှိ အပေါင်ဆုံး CheckBox ကို on လိုက်သည်နှင့် Mortgage Receive တွင်သတ်မှတ်ခဲ့သော အမျိုးအစား၊ အမျိုးအမည်၊ ရွှေချိန်၊ ပေးငွေ ကျလာမည်။
  • အပေါင်ဆုံးစာရင်းသတ်မှတ်ပြီးပါက Save Button ကိုနှိပ်၍ သိမ်းဆည်းထားနိုင်ပပါသည်။
  • Mortgage Disable Form ကိုအသုံးပြုပြီးပါက Close Buttonကိုနှိပ်၍ ပိတ်နိုင်ပါသည်။
